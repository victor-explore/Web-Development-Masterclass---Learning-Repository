Dark Mode Example

About Dark Mode

Dark mode reduces eye strain in low-light conditions and can save battery life on OLED displays. This example demonstrates a simple implementation of dark mode using Tailwind CSS utility classes.

Click the "Toggle Dark Mode" button in the header to switch between light and dark modes.

Automatic Dark Mode

Tailwind CSS supports automatic dark mode based on user's system preferences using the 'prefers-color-scheme' media query.

Manual Toggle

You can also implement a manual toggle like this example, which adds a 'dark' class to the HTML element.

Code Example 

<div class="bg-white dark:bg-gray-800 text-gray-800 dark:text-white">
  This element has different styling in dark mode
</div>

The dark: prefix applies styles only when dark mode is active.
