What is Bootstrap?

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains HTML, CSS, and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

Quick Tip: Bootstrap uses a 12-column grid system for creating page layouts.

Getting Started

To use Bootstrap in your project, you can include it via CDN:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">

Key Bootstrap Components

Grid System

Bootstrap's grid system uses a series of containers, rows, and columns to layout and align content.

<div class="row">
  <div class="col-md-6">Column 1</div>
  <div class="col-md-6">Column 2</div>
</div>
Buttons

Bootstrap includes several predefined button styles, each serving its own semantic purpose.

<button class="btn btn-primary">Primary</button>
Alerts

Alerts provide contextual feedback messages for typical user actions.

This is a success alert!
<div class="alert alert-success">
  This is a success alert!
</div>
Forms

Bootstrap provides several form control styles and layout options.

<input type="email" class="form-control">
Navbar

Bootstrap's responsive navigation header includes support for branding, navigation, and more.

  • Pricing

Key Takeaways

  • Bootstrap uses classes like form-control, form-label, and form-select for form elements
  • The grid system (row and col-*) helps create responsive layouts
  • Bootstrap's utility classes like mb-4 (margin-bottom) help with spacing
  • Components like form-check provide styled form controls

Bootstrap Utilities

Bootstrap includes dozens of utilities - helper classes for common CSS properties:

Spacing

<div class="mt-3">Margin top 3</div>
<div class="p-4">Padding 4</div>
<div class="mb-5">Margin bottom 5</div>

Colors

<div class="text-primary">Primary text</div>
<div class="bg-success">Success background</div>
<div class="border border-danger">Danger border</div>

Display

<div class="d-flex">Flexbox container</div>
<div class="d-none d-md-block">Hidden on small screens</div>

Text

<p class="text-center">Centered text</p>
<p class="fw-bold">Bold text</p>
<p class="text-truncate">Truncated text...</p>
